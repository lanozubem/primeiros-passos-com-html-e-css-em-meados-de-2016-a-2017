Hard reset para o MOTO G XT1556

VOLTAR

Não importa quão dificil foi o padrão que pôs em seu dispositivo, com esse procedimento o mesmo sairá.
i Lembre-se que todos os dados do aparelho serão perdidos.



  1. - Desligue o aparelho.
  2. Pressione simultanêamente as teclas POWER + VOL - por aproximadamente 3s e solte o botão Power.
  3. No menu fastboot desça com a tecla Vol - até RECOVERY e pressione a tecla VOL + para confirmar.
  4. Desça até WIPE DATA/FACTORY RESET, confirme com o botão Power.
  5. Agora vá até YES___DELETE ALL USER DATE e confirme novamente.
  6. Ao voltar a tela de Recovery confirme em Reboot System Now para poder reiniciar o aparelho.
  7. Após aguardar o dispositivo iniciar o mesmo já estará sem o padrão.
