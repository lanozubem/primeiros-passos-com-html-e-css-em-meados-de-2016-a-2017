Instalar stock rom original de fábrica no Moto G XT1032

VOLTAR

Se por acaso seu dispositivo entrou em Loop infinito, boot infinito ou simplesmente deseja retornar a versão original do sistema esse é o tutorial certo que você tanto procurava. Esse procedimento servirá não só para voltar a uma versão de sistema anterior mas como resolver erros de sistema como é o caso do irritante " pop up " que apresenta-se como O Processo.System parou e também para remover virus.

Para esse tutorial você irá prescisar de - Adb + Fastboot Firmware Moto G XT1032

Então vamos deixar de enrolação e partir para o tutorial!


    No PC

  1. Primeiramente baixe os 2 arquivos listados acima e os extraia em algum diretório do seu Pc.
  2. Após extraí-los abra a pasta Adb + Fastboot e copie os arquivos que nela contém.
  3. Os arquivos que acabamos de copiar, cole na pasta soinroms_MOTOG_XT1032.
  4. Feito o passo anterior, na mesma pasta que acabamos de colar, clique ao mesmo tempo na segunda tecla do mouse e na tecla Shift do pc então escolha Abrir janela de comando aqui.
  5. Feito o passo anterior aparecerá uma janela preta _ CMD do windows, agora é hora de partir para o celular.

    6. No Celular

  6. - Desligue o aparelho.
  7. Pressione simultanêamente as teclas POWER + VOL - por aproximadamente 3s e solte o botão Power para entrar no menu Fastboot.
  8. Conecte o Moto G ao pc.
  9. Agora informe essas linhas de comando na janela CMD que abrirá em seu Pc.

    10. mfastboot.exe flash partition gpt.bin

    mfastboot.exe flash motoboot motoboot.img

    mfastboot.exe flash logo logo.bin

    mfastboot.exe flash boot boot.img

    mfastboot.exe flash recovery recovery.img

    mfastboot.exe flash system system.img_sparsechunk.0

    mfastboot.exe flash system system.img_sparsechunk.1

    mfastboot.exe flash system system.img_sparsechunk.2

    mfastboot.exe flash system system.img_sparsechunk.3

    mfastboot.exe flash modem NON-HLOS.bin

    mfastboot.exe erase modemst1

    mfastboot.exe erase modemst2

    mfastboot.exe flash fsg fsg.mbn

    mfastboot.exe erase cache

    mfastboot.exe erase userdata

    mfastboot reboot

    Após esta última linha de comando seu dispositivo já irá reiniciar, então vá beber um cafezinho enquanto o mesmo não inicia.
