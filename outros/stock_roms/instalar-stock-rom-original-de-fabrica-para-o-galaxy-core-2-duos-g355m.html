Instalar stock rom original de fábrica Android 4.2.0 Jelly Bean para o Galaxy Core 2 Duos G355M

VOLTAR

Procedimento de reinstalação de firmware, disponivel para o dispositivo Samsung Galaxy Core 2 Duos, voltando a versão original do sistema de forma rápida e sem nenhuma complicação.

Para esse tutorial você irá prescisar de

Odin Firmware para o Galaxy Core 2 Duos G355M Drivers

A coversa está boa mas o que é bom mesmo é o tutorial, então, vamos lá!

    No PC

  1. Você deve baixar os 3 arquivos listados acima, no seu pc extraia os arquivos "www.soinroms.com_SAM_G355M.rar", "Odin3-v1.85_2_www.soinroms.com.rar" e instale os drivers USB.
  2. Vamos utilizar o programa odin para realizar a instalação da firmware, então vá no diretório onde foi realizado a extração e o abra.
  3. Após abrir o programa Odin vamos partir para o telefone.

    4. No Celular

  4. - Desligue o aparelho.
  5. Pressione ao mesmo tempo as teclas HOME, VOL - e POWER e solte a tecla POWER quando a tela de Download aparecer.
  6. Agora pressione a tecla de VOL + para confirmar.
  7. Conecte o G355M ao pc e note que no programa Odin a janela ID:COM ficará amarela indicando que o dispositivo foi reconhecido.
  8. No odin clique em PDA e escolha a firmware que acabamos de extrair.
  9. Ao selecionar a firmware G355MUBU0ANF7_G355MZTO0ANF1_G355MUBU0ANF7_HOME.tar.md5 você irá pressionar em Start, pronto agora só é aguardar o processo de instalação terminar.
  10. Após tudo isso o dispositivo irá reiniciar, enquanto o mesmo não inicia, descanse um pouco ^-^.

    11. OBSERVAÇÃO: Se ao realizar os passos acima e a instalação resultar em erro faça o seguinte.
    Vá no diretório onde foi realizado o download da firmware e renomei a extensão .MD5 para .TAR ficando o resultado final assim...
    "G355MUBU0ANF7_G355MZTO0ANF1_G355MUBU0ANF7_HOME.tar"

  11. Para finalizarmos, recomendamos que faça um procedimento de Hard reset para que assim todas as funcionalidades funcionem bem. Se você não sabe como realiza-lo, clique Aqui para prosseguir.
