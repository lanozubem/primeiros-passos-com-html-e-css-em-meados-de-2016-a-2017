Instalar Stock rom original de fabrica para o LG L40 D175F na versão Android 4.1.2 KITKAT

VOLTAR

Veja agora como reinstalar a versão original de fábrica para o LG L40 e resolver erros como: 'O Processo.System Parou, travamentos, lentidões loop infinito' ou se você instalou uma Custom rom e não se agradou, com esse procedimento retornarás ao que era antes. Chega de vírus, propagandas indesejáveis, custom recovery e vários outros.

Para esse tutorial você irá prescisar do ::

LG Flash Tool 2014 Firmware LG L40 D175F Drivers

A coversa está boa mas o que é bom mesmo é o tutorial, então, vamos lá!


    No PC

  1. Após ter baixado os 3 arquivos listados acima extraia o "LG Flash Tool 2014.rar"
  2. Extraia e instale os drivers USB "LGMobileDriver_WHQL_Ver_4.2.0.rar", para que assim o pc reconheça seu aparelho.
  3. Vamos utilizar o programa Lg Flash Tool 2014 para realizar a instalação da firmware, então vá no diretório onde foi realizado o download e o execute.
  4. Após a instalação do Flash Tool vamos partir para o telefone.
  5. Mas antes, abra o programa e conecte o cabo USB ao PC.

    6. No Celular

  6. - Desligue o aparelho.
  7. Vamos por o L40 em modo Download.
  8. Com o aparelho desligado pressione ao mesmo tempo a tecla Vol + e conecte o cabo usb até que apareça a tela de firmware update, há lembrando que o mesmo já deve está conectado ao computador.
  9. No programa Flash tool clique no icone da "Pasta" para escolher a firmware 'www.soinroms.com_D175f10e_00.kdz' .
  10. Após escolher o arquivo mencionado clique em 'CSE FLASH' e em 'START' e aguarde esta tela.
  11. Clique em Ok, não se preocupe quanto ao país e a linguagem, agora aquarde o processo terminar.
  12. Após tudo isso o dispositivo irá reiniciar com um sistema novinho, enquanto o mesmo não inicia, vá ver aquela pessoa que é um deleite para seus olhos <3<3>.
