Instalar stock rom original de Fábrica Android 6.0 Marshmallow no MOTO G XT1069 DSTV

VOLTAR

Se por o seu dispositivo entrou em Loop infinito, boot infinito ou simplesmente deseja retornar a versão original do sistema esse é o tutorial certo que você tanto procurava. Esse procedimento servirá não só para voltar a uma versão de sistema anterior mas como resolver erros de sistema como é o caso do irritante 'pop up' que apresenta-se como 'O Processo.System parou' e também para remover virus.

Para esse tutorial você irá prescisar do :

FASTBOOT Firmware para o MOTO G XT1069 DSTV Drivers

A coversa está boa mas o que é bom mesmo é o tutorial, então, vamos lá!


    No PC

  1. Primeiramente baixe os 3 arquivos listados acima e os extraia em algum diretório do seu Pc.
  2. Após extraí-los abra a pasta adb+mfastboot_www.soinroms.com e copie os arquivos que nela contém.
  3. Cole os arquivos que acabamos de copia na pasta 'www.soinroms.com -XT1069_TITAN_RETBR_DSTV_6.0.rar'.
  4. Feito o passo anterior, na mesma pasta que acabamos de colar, clique ao mesmo tempo na tecla direita do mouse e na tecla Shift do pc, então escolha "Abrir janela de comando aqui".
  5. Feito o passo anterior aparecerá uma janela com o prompt de comando _ CMD do windows, agora é hora de partir para o aparelho.

    6. No Celular

  6. - Desligue o aparelho.
  7. Pressione simultanêamente as teclas POWER + VOL - por aproximadamente 3s e solte o botão Power para entrar no menu Fastboot.
  8. Conecte o MOTO G ao pc.
  9. Agora informe essas linhas de comando na janela CMD que abrirá em seu Pc.

    1. mfastboot.exe flash bootloader bootloader.img

    2. mfastboot.exe flash partition gpt.bin

    3. mfastboot.exe flash motoboot motoboot.img

    4. mfastboot.exe flash logo logo.bin

    5. mfastboot.exe flash boot boot.img

    6. mfastboot.exe flash recovery recovery.img

    7. mfastboot.exe flash system system.img_sparsechunk.0

    8. mfastboot.exe flash system system.img_sparsechunk.1

    9. mfastboot.exe flash system system.img_sparsechunk.2

    10. mfastboot.exe flash system system.img_sparsechunk.3

    11. mfastboot.exe flash modem NON-HLOS.bin

    12. mfastboot.exe erase modemst1

    13. mfastboot.exe erase modemst2

    14. mfastboot.exe flash fsg fsg.mbn

    15. mfastboot.exe erase cache

    16. mfastboot.exe erase userdata

    17. mfastboot reboot

  10. Após essa última linha de comando o aparelho reiniciará automaticamente.
  11. O processo de inicialização poderá durar entre 10 à 15 minutos, enquanto o mesmo não inicia que tal adicionar o SO. IN ROMS a sua barra de favoritos? :) .
