Instalar stock rom original de fábrica Samsung J1 Ace J111F

VOLTAR

Procedimento de reinstalação de firmware, disponivel para o dispositivo Samsung J111F, voltando a versão original do sistema de forma rápida e sem nenhuma complicação.

Para esse tutorial você irá prescisar de - Odin Firmware SM-J111F

A coversa está boa mas o que é bom mesmo é o tutorial, então, vamos lá!


    No PC

  1. Você deve baixar os 2 arquivos listados acima, no seu pc extraia apenas o segundo arquivo SM-J111F.rar
  2. Vamos utilizar o programa odin para realizar a instalação da firmware, então vá no diretório onde foi realizado o download e o execute | instale o odin.
  3. Após a instalação do Odin vamos partir para o telefone.
  4. Mas antes, abra o programa.

    5. No Celular

  5. - Desligue o aparelho.
  6. Pressione ao mesmo tempo as teclas HOME, VOL - e POWER e solte a tecla POWER quando a tela de Download aparecer.
  7. Agora pressione a tecla de VOL + para confirmar.
  8. Conecte o J1 ao pc, note que no programa Odin a janela ID:COM ficará amarela confirmando que o dispositivo foi reconhecido.
  9. No odin clique em PDA e escolha a firmware que acabamos de extrair.
  10. Ao selecionar a firmware J111F.tar você irá pressionar em Start, pronto agora só é aguardar o processo de instalação terminar.
  11. Após tudo isso o dispositivo irá reiniciar, enquanto o mesmo não inicia, vá ler uma crônica cronática |+_+|.
  12. Para finalizarmos, recomendamos que faça um procedimento de Hard reset para que assim todas as funcionalidades funcionem bem. Se você não sabe como realiza-lo, clique Aqui para prosseguir.
