Instalar stock rom original de fábrica Android 5.1.1 Lollipop no Samsung Galaxy J2 J200M

VOLTAR

Procedimento de reinstalação de firmware, disponivel para o dispositivo Samsung Galaxy J2, voltando a versão original do sistema de forma rápida e sem nenhuma complicação. Com isso, problemas como: Incapacidade de iniciar - Loop infinito, Travamentos, lentidões, Virus - aplicativos instalam automaticamente - Propagandas exessivas, custom-recovery's e vários outros não irão mas o pertubarem.

Para esse tutorial você irá prescisar do ::

Odin Firmware para o Galaxy J2 SM-J200M Drivers

A coversa está boa mas o que é bom mesmo é o tutorial, então, vamos lá!


    No PC

  1. Você deve baixar os 3 arquivos listados acima, no seu pc extraia os arquivos "www.soinroms.com_Samsung_Galaxy_J2_SM-J200M.rar", "Odin3-v1.85_2_www.soinroms.com.rar" e instale os drivers USB.
  2. Vamos utilizar o programa odin para realizar a instalação da firmware, então vá no diretório onde foi realizado a extração e o abra.
  3. Após abrir o programa Odin vamos partir para o telefone.

    4. No Celular

  4. - Desligue o aparelho.
  5. Pressione ao mesmo tempo as teclas HOME, VOL - e POWER e solte a tecla POWER quando a tela de Download aparecer.
  6. Agora pressione a tecla de VOL + para confirmar.
  7. Conecte o J2 ao pc, note que no programa Odin a janela ID:COM ficará amarela indicando que o dispositivo foi reconhecido.
  8. No odin clique em PDA e escolha a firmware que acabamos de extrair.
  9. Ao selecionar a firmware "J200MUBU1AOK2_J200MZTA1AOK2_J200MUBU1AOK2_HOME.tar" você irá pressionar em Start e aguardar o processo de instalação terminar.
  10. Após tudo isso o dispositivo irá reiniciar, enquanto o mesmo não inicia, vá ler um Romance <3.
  11. Para finalizarmos, recomendamos que faça um procedimento de Hard reset para que assim todas as funcionalidades funcionem bem. Se você não sabe como realiza-lo, clique Aqui para prosseguir.
