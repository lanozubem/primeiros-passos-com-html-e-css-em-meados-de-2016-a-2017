Instalar stock Original de Fábrica Android 6.0.1 Marshmallow para o GALAXY S5 SM-G900MD

VOLTAR

Procedimento de reinstalação de firmware, disponivel para o dispositivo GaLAXY S5 SM-G900MD, voltando a versão original do sistema de forma rápida e sem nenhuma complicação. Odin Firmware para o GALAXY S5 SM-G900MD Drivers

A coversa está boa mas o que é bom mesmo é o tutorial, então, vamos lá!


    No PC

  1. Você deve baixar os 3 arquivos listados acima, no seu pc extraia os arquivos "www.soinroms.com_G_S5_SM-G900MD_6.0.1.rar", "Odin3-v1.85_2_www.soinroms.com.rar" e instale os drivers USB.
  2. Vamos utilizar o programa odin para realizar a instalação da firmware, então vá no diretório onde foi realizado a extração e o abra.
  3. Após abrir o programa Odin vamos partir para o telefone.

    4. No Celular

  4. - Desligue o aparelho.
  5. Pressione ao mesmo tempo as teclas HOME, VOL - e POWER e solte a tecla POWER quando a tela de Download aparecer.
  6. Agora pressione a tecla de VOL + para confirmar.
  7. Conecte o Galaxy S5 ao cabo usb, note que no programa Odin a janela ID:COM ficará amarela indicando que o dispositivo foi reconhecido.
  8. No odin clique em PDA e escolha a firmware que acabamos de extrair.
  9. Ao selecionar a firmware "G900MDUBS1CPJ2_G900MDZTO1CPE1_G900MDUBU1CPE1_HOME.tar.md5" você irá pressionar em Start, pronto agora só é aguardar o processo de instalação terminar.
  10. Após tudo isso o dispositivo irá reiniciar, enquanto o mesmo não inicia, vá "espiá" seu smart ligar 00.).
